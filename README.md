# BookmarksAPI

An API for all your bookmarks.

## Local Deployment
1. clone the repo and install the requirements.
2. Setup these enivronment variables :
```
export SECRET_KEY="ThisisT0pS3cr3t"
export JWT_SECRET_KEY="D0ntR3v3alit"
```
3. Run the server :
```
flask run
```
